About Project

This project helped me to improve the base of HTML, CSS and PHP.
The project target was to create new Brand “ECODEWS” and bring new visitors to the website; using popular traffic sources such as Youtube, Facebook and Google(organic search). It let me see more of the SEO side of Web Development, bringing new visitors to the website and following their behaviours using Google Analytics. Ecodews reached a maximum of 77 website daily visitors.
This projects helped me to understand how important is to keep code tidy and organized. When project scaled it get harder to organize and manage code. That's when I discovered MVC pattern and started to explore Frameworks.
